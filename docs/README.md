## üìä The Complete Pipeline

```
External DBs ‚Üí Airbyte ‚Üí Spark ‚Üí DBT ‚Üí Feast ‚Üí MLOps
    ‚Üì           ‚Üì         ‚Üì       ‚Üì      ‚Üì       ‚Üì
 Sources     Bronze    Silver   Gold  Features Models
```

### Data Layers

1. **Bronze** (Raw) - Airbyte ingestion from sources
2. **Silver** (Cleaned) - Spark data quality and validation
3. **Gold** (Aggregated) - DBT business logic and metrics
4. **Features** (ML-Ready) - Feast feature views
5. **Models** (Deployed) - MLflow trained models

## üéØ Common Tasks

### Create Data Source

```bash
curl -X POST http://localhost:8000/datasource \
  -H "Content-Type: application/json" \
  -d '{"source_type": "postgres", "name": "my_db", ...}'
```

### Transform Data

```bash
curl -X POST http://localhost:8000/spark/transform \
  -H "Content-Type: application/json" \
  -d '{"sql_query": "SELECT * FROM bronze.table", ...}'
```

### Register Features

```bash
curl -X POST http://localhost:8000/feast/features \
  -H "Content-Type: application/json" \
  -d '{"name": "features", "entities": ["id"], ...}'
```

### Train Model

```bash
curl -X POST http://localhost:8000/mlops/training/upload \
  -H "Content-Type: application/json" \
  -d '{"script_content": "<base64>", ...}'
```

### Make Predictions

```bash
curl -X POST http://localhost:8000/mlops/inference \
  -H "Content-Type: application/json" \
  -d '{"model_name": "model", "inputs": {...}}'
```
